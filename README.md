# Telephone call based 2FA for MediaWiki
TelOTP lets users set up 2FA by providing their telephone number. When the user goes to log in, they will receive a call with a code they need to enter. 
This module requires and supports one of several external telephony engines. Asterisk is recommended. You will also need a trunk to an external provider to send calls to the PSTN. You can use one of many VoIP providers, or a hardware FXO or PRI gateway for POTS or PRI lines.
